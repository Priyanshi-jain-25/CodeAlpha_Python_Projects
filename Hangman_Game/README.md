# Tech Hangman Game

A simple Python-based Hangman game where players guess hidden programming language names letter by letter.

---

## Features

- Random word selection
- Attempt tracking
- Input validation
- Interactive gameplay
- Win and lose conditions

---

## Technologies Used

- Python
- Random Module
