# CodeAlpha Python Internship Projects

This repository contains 4 Python projects completed as part of the CodeAlpha Python Programming Internship.

These projects demonstrate:

* Python fundamentals
* Problem solving
* Automation
* File handling
* User interaction
* Logic building

---

# Projects Included

## 1. Hangman Game

A programming-language-themed Hangman game where the user guesses letters to identify the correct word.

### Features

* Random word selection
* Input validation
* Attempt tracking
* Win and lose conditions
* Programming language theme

### Technologies Used

* Python
* Random Module

---

## 2. Stock Portfolio Tracker

A Python application that calculates the total investment value of selected stocks.

### Features

* Stock price dictionary
* Investment calculation
* Total portfolio value
* File report generation
* User-friendly interaction

### Technologies Used

* Python
* File Handling

---

## 3. Task Automation With Python Scripts

A Python automation script that automatically moves JPG image files from one folder to another.

### Features

* Automatic file movement
* JPG file filtering
* Folder management
* Error handling
* File counting system

### Technologies Used

* Python
* OS Module
* Shutil Module

---

## 4. Smart Python Chatbot

A simple intelligent chatbot that responds to greetings, time, date, motivation, and user queries.

### Features

* Greeting system
* Time and date feature
* Motivation and study tips
* Cultural greetings
* Interactive responses

### Technologies Used

* Python
* Datetime Module
